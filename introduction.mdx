## About This Project

The MilesONerd Website is a personal portfolio and blog platform for Enzo Fuke, showcasing expertise in Data Science, Machine Learning, and Full Stack Development. Built with modern web technologies, it provides a clean, responsive interface with custom animations and a dark theme.

## Key Features

<AccordionGroup>
  <Accordion title="Personal Portfolio Homepage" icon="house">
    A stunning hero section with gradient background, animated terminal, and professional introduction showcasing skills and experience
  </Accordion>
  
  <Accordion title="About Page" icon="user">
    Detailed professional background highlighting expertise in Python, SQL, Go, R, JavaScript, and more, with site history timeline
  </Accordion>
  
  <Accordion title="Blog System" icon="newspaper">
    Markdown-based blog with automated post generation using shell scripts and dynamic post listing
  </Accordion>
  
  <Accordion title="Contact Form" icon="envelope">
    Integrated Formspree contact form with privacy-focused email handling using temporary Guerrilla Mail addresses
  </Accordion>
  
  <Accordion title="Responsive Design" icon="mobile">

  </Accordion>
  
  <Accordion title="Dark Theme" icon="moon">

  </Accordion>
  
  <Accordion title="Social Integration" icon="share-nodes">

## Technology Stack

The website is built with:

- **HTML5** - Semantic markup structure
- **Tailwind CSS** - Utility-first CSS framework via CDN
- **JavaScript** - Interactive functionality and animations
- **Markdown** - Blog post content format
- **Shell Script** - Automated build processes
- **Netlify** - Hosting and continuous deployment
- **Formspree** - Contact form backend
